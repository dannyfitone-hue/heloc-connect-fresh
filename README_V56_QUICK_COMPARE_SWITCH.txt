V56 Quick Compare Switch

Changes made:
- Added a premium Quick Compare button in the estimated preview area.
- If the client is viewing HELOC numbers, the button says: Show Me Refinance Numbers.
- If the client is viewing Refinance numbers, the button says: Show Me HELOC Numbers.
- The switch reuses the same address, home value, existing loan balances, requested cash, and rate fields.
- If the target option needs missing information, the page now highlights the required fields in red and scrolls the client directly to the first missing input.
- Missing refinance fields include current loan interest rate and each additional loan rate when multiple existing loans are selected.
- No portal, backend, SMS, status, owner dashboard, or lender dashboard logic was changed.
